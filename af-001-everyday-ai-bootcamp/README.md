# Purpose

AF-001 — Everyday AI Bootcamp is a practical, beginner-focused AI course designed to help ordinary people safely, effectively, and thoughtfully use modern AI tools in personal and professional settings.

The course focuses on:
- practical AI usage
- structured thinking
- responsible interaction
- communication clarity
- human judgment
- workflow improvement

This course is intentionally:
- lightweight
- globally accessible
- tool-neutral
- beginner-friendly
- practical rather than theoretical

---

# Core Principle

> AI should assist human capability, not replace human judgment.
