Status: ACTIVE DEVELOPMENT

# Purpose

This document describes the operational setup requirements for AF-001 — Everyday AI Bootcamp.

The course is intentionally designed to require minimal setup and minimal technical overhead.

---

# Minimum Requirements

Required:
- access to a conversational AI system
- ability to read Markdown or plain text files

Examples of supported conversational AI systems may include:
- ChatGPT
- Claude
- Gemini
- Grok
- similar AI platforms

No programming environment is required.

No local installation is required.

No special tooling is required.

---

# Recommended Environment

Recommended:
- stable internet connection
- desktop or laptop computer
- ability to open multiple browser tabs or windows

Optional:
- note-taking application
- text editor
- personal notebook

---

# Recommended Learner Approach

AF-001 is designed to work best when the learner:
- experiments actively
- compares outputs
- refines prompts
- reflects on results
- proceeds at a comfortable pace

The course favors:
- practical experimentation
- lightweight structure
- iterative learning

---

# Operational Philosophy

AF-001 intentionally avoids:
- heavy onboarding
- account requirements
- enterprise tooling
- mandatory software installation
- complex orchestration

The setup process should remain:
- understandable
- lightweight
- globally accessible
- survivable at low technical skill levels
