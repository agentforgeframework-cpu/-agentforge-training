# Lesson Purpose

This lesson introduces the learner to practical AI prompting fundamentals.

The goal is not to memorize “magic prompts.”

The goal is to learn how better structure improves AI interaction quality.

Learners should finish this lesson with:
- confidence interacting with AI systems
- awareness of prompt structure
- understanding that AI output quality often depends on input quality
- practical experience refining prompts

# SECTION 1/5 — What Is a Prompt?

A prompt is the instruction or input given to an AI system.

Examples:
- a question
- a request
- instructions
- a task description
- a conversation

Prompts may be:
- short
- detailed
- conversational
- structured

AI systems often respond better when the request is:
- clear
- specific
- organized

---

# SECTION 2/5 — Baseline Prompt Exercise

Start with a very simple request.

Example:

> Tell me about camping.

Observe:
- What did the AI assume?
- What information was missing?
- Was the response useful?
- Was it too broad?

There is no “wrong” result here.

The purpose is observation.

---

# SECTION 3/5 — Structured Prompt Exercise

Now provide more structure.

Example:

> I am a beginner camper preparing for a weekend camping trip with my family in cool weather. Create a simple packing checklist with short explanations.

Compare this result to the earlier example.

Observe:
- clarity
- usefulness
- tone
- organization
- relevance

---

# SECTION 4/5 — Refinement Loop

One of the most important AI skills is refinement.

Good AI usage is often iterative.

Simple refinement examples:
- “Make this shorter.”
- “Explain this more simply.”
- “Add examples.”
- “Organize this into bullets.”
- “Rewrite this for a teenager.”
- “Rewrite this professionally.”

Try refining one of your earlier prompts.

Observe how the output changes.

---

# SECTION 5/5 — Human-in-Command Reminder

AI can:
- assist
- organize
- summarize
- generate ideas

But the human remains responsible for:
- judgment
- verification
- decisions
- final use

AI output should support thinking, not replace it.
