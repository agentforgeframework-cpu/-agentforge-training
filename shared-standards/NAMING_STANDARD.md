# Naming Standard

## General Rules

- lowercase only
- hyphen-separated names
- no spaces
- avoid unnecessary abbreviations
- prioritize readability

## Folder Naming

Correct:
- iot-aiot-for-sas-programmers
- shared-standards
- companion-papers

Incorrect:
- IoT AIOT Stuff
- Lesson1
- tempFiles

## File Naming

Prefer:
- COURSE_STANDARDS.md
- LESSON_TEMPLATE.md
- RELEASE_MODEL.md

## Lesson Folder Naming

Use:
- 01-simple-video-and-images
- 02-events-schemas-exception-handling
