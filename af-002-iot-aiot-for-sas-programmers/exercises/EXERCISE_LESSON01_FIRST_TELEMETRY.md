# Task

Create or use a telemetry CSV and import it into SAS.

# Steps

1. Run the webcam telemetry Python script, or use the sample CSV.
2. Confirm the CSV has rows and timestamps.
3. Import the CSV into SAS.
4. Print the first 10 rows.
5. Run a simple summary.
6. Create one simple event flag.

# Success

You have succeeded when you can say:

> I generated or used telemetry data and analyzed it in SAS.
