# Current Governance Documents

## AF002-GOV-001_Controlled_Evidence_Doctrine.md

Purpose:
Establishes operational controls for instructional evidence used during runtime walkthroughs and operational lessons.

Key principle:

Operational evidence used during instructional runtime must be:
- approved
- reviewable
- reproducible
- stable across sessions

This doctrine specifically prohibits:
- dynamically generated runtime imagery
- uncontrolled visual evidence artifacts
during operational instructional sessions.
