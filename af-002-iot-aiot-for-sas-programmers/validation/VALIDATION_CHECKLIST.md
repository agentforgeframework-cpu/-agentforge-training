# Onboarding Check

- [ ] README explains what AF-002 is.
- [ ] QUICKSTART shows where to begin.
- [ ] SETUP lists required assumptions clearly.
- [ ] Learner can find Lesson 00.

# Lesson 00 Check

- [ ] Python check is understandable.
- [ ] SAS check is understandable.
- [ ] Webcam or image fallback is understandable.
- [ ] Learner knows when to begin Lesson 01.

# Lesson 01 Check

- [ ] Learner can create or use a telemetry CSV.
- [ ] Learner can import the CSV into SAS.
- [ ] Learner can see output.
- [ ] Learner can explain why this is telemetry.

# Portability Check

- [ ] ChatGPT can guide the course.
- [ ] Claude can guide the course.
- [ ] LOAD files do not become learner-facing clutter.
- [ ] Setup does not assume hidden tools.

# 2 AM Engineer Check

- [ ] A tired SAS programmer could start successfully with minimal help.
- [ ] The course avoids unnecessary theory before first success.
- [ ] The structure remains understandable.
