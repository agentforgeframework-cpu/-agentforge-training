# QUICKSTART — AF-002 IoT/AIoT for SAS Programmers

## Purpose

This QUICKSTART helps a human learner understand what the course does and where to begin.

This is the human entry point.

---

## What You Will Build First

Your first working success will be:

```text
webcam → telemetry → CSV → SAS analysis
```

In plain language:

1. Use a webcam as a simple sensor.
2. Capture basic operational observations.
3. Save those observations as a CSV file.
4. Import the CSV into SAS.
5. Inspect the data using normal SAS workflows.

You do not need industrial IoT hardware to begin.

Your laptop can become the first sensor.

---

## Why This Matters for SAS Professionals

SAS programmers already work with:

- structured data
- timestamps
- events
- trends
- thresholds
- summaries
- operational reporting
- decision support

IoT becomes less mysterious when it becomes timestamped operational data.

---

## Before You Begin

You should have access to:

- a Windows PC
- a webcam
- internet access
- an AI assistant
- Python or the ability to install Python
- SAS access, such as SAS OnDemand for Academics or another SAS environment

The course is designed to use free and readily available tools where practical.
