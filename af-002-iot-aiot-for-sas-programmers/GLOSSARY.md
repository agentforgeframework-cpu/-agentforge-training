# AF-002 Glossary Additions

## Bounding Box

A rectangular coordinate region used to represent the detected position
of an object within an image or frame.

Bounding boxes are commonly represented using:
- x1
- y1
- x2
- y2

Bounding boxes transform visual detections into structured telemetry
suitable for operational investigation using SAS.

## Spatial Telemetry

Structured telemetry representing the position, movement, occupancy,
or region interaction of objects within a visual environment.

## Region Analysis

Operational investigation using predefined spatial zones such as:
- restricted
- loading
- pedestrian
- staging
