# Lesson 04 SAS Programs

Place these files in the AF-002 `/sas` folder.

Files:

- `AF002_LESSON_04_OPERATIONAL_TELEMETRY.sas`
  - Core Lesson 04 static SAS program.
  - Creates the teaching data and produces the primary outputs.

- `AF002_LESSON_04_VISUAL_REVIEW.sas`
  - Optional visual review program.
  - Run after the core program.
  - Useful for testing which plots should remain in the lesson.

Recommended first test:

1. Open SAS OnDemand for Academics.
2. Run `AF002_LESSON_04_OPERATIONAL_TELEMETRY.sas`.
3. Confirm the log shows `WORK.AF002_L04_WAREHOUSE has 6 observations`.
4. Review whether the output supports the Lesson 04 story:
   - one company
   - three operating phases
   - operational telemetry credibility
   - SAS as practical meeting preparation
